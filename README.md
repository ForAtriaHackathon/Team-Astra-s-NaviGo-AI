## 🚀 About The Project

NaviGo AI is an intelligent citizen-assistance platform that democratizes access to public services. It helps users discover government schemes, verify eligibility, understand legal requirements, and seamlessly navigate applications with AI-powered step-by-step guidance. 

By combining autonomous Multi-Agent AI architecture and dynamic knowledge retrieval, NaviGo AI dramatically reduces bureaucratic complexity and makes government services accessible to everyone.

## ✨ Key Features

- **🎯 Smart Scheme Discovery**: Instantly find relevant government schemes through natural conversation.
- **✅ Automated Eligibility Checker**: Chat with our AI to instantly know if you qualify for a scheme without reading 50-page documents.
- **📄 AI Document Understanding**: Upload documents and let the AI verify if they meet application requirements to prevent rejections.
- **🗺️ Step-by-Step Roadmaps**: Get a personalized, actionable 1-2-3 guide on exactly how to apply.
- **🌐 Multilingual Support**: Talk to the assistant in multiple languages for maximum accessibility.

## 🛠️ Technology Stack

- **Frontend:** Vanilla HTML5, CSS3 (Glassmorphism UI), JavaScript
- **Backend:** Python, FastAPI, Uvicorn
- **AI Brain:** Google Gemini 1.5 Flash (Multi-Agent Architecture)
- **Deployment:** Render

## 💻 Local Installation

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ForAtriaHackathon/Team-Astra-s-NaviGo-AI.git
   cd Team-Astra-s-NaviGo-AI
   ```

2. **Set up the backend:**
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

3. **Set your Environment Variables:**
   ```bash
   # Windows (Powershell)
   $env:GEMINI_API_KEY="your-google-gemini-api-key"
   
   # Linux/Mac
   export GEMINI_API_KEY="your-google-gemini-api-key"
   ```

4. **Run the server:**
   ```bash
   python main.py
   ```
   *The server will start on `http://localhost:8000` and automatically serve the frontend.*
